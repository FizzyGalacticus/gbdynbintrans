# gbdynbintrans
Extremely basic GameBoy ROM dynamic binary translator.

## Dev Tools:
[GBDK - GameBoy Development Kit](https://sourceforge.net/projects/gbdk/)

## Compile .s assembly  files 
```
gbdk/bin/lcc -o <outputfilename> <inputfilename>
```
